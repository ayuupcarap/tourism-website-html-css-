Website User Guide

Welcome to the Somalia Tourism Board website. This page explains how to use every page of the website.

🏠 Home Page

The Home page introduces Somalia Tourism Board. You can read about Somalia's tourism, view featured attractions, and use the navigation menu to visit other pages.

🖼 Gallery Page

The Gallery page displays beautiful tourist attractions. It also includes a tourism information table showing locations, categories and ratings.

📝 Booking Page

Fill in the booking form with your personal information, choose a tour package, select your hobbies, and submit your booking request.

📖 About Page

The About page contains information about the student, the project, and the purpose of the website.

🧭 Navigation Menu

Use the navigation menu at the top of every page to move between Home, Gallery, Booking, Help and About pages.

🖱 Tips

Hover your mouse over buttons and links to see visual effects. Images, tables and forms are responsive and work on mobile devices.

Instructions

  1. Open the Home page first.
  2. Visit the Gallery page to view tourist attractions.
  3. Read the tourism information table.
  4. Open the Booking page.
  5. Complete all required fields.
  6. Click the Book Now button.
  7. Visit the About page to learn about the project.
